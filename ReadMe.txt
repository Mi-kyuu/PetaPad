0.初期設定
	初回起動時、同ディレクトリにpeta.bmpとsettingsファイルを生成します。




1.使用方法
	(重要)PetaPad.exeとPetaView.exeは同じディレクトリに配置してください。

	付箋を新規作成、編集するときはPetaPad.exeを起動してください。編集した後、"Peta"ボタンを押すことで同ディレクトリのPetaView.exeが起動、張り付きます。
	付箋(PetaView.exe)を終了する場合は、付箋上にマウスカーソルを合わせてEscキーを押すか、	付箋右上の×ボタンを押してください。
	(×マークを押すときは裏のオブジェクトに注意してください)

	PetaView.exeのショートカットを作成しスタートアップフォルダ("C:\Users\[ユーザー名]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup", Win10の場合)に投げ込むと起動時に毎回立ち上がります。




2.編集画面説明(PetaPad.exe)
	キャンバス(300×200)
		クリックすることでお絵描きできます。
		
	[操作説明]
	①Reset(上部)
		ボタンを押すことで指定の色でキャンバスを初期化します。
	②Weight(右上部)
		ペンの太さ調節ができます。1×1pxから4×4pxまでの4段階。
	③Pen Color(右上部)
		ペンの色を変更できます。この色はテキストにも反映されます。
	④Text
		入力フォームで文字入力し、Enterボタンを押す、または[Alt+Enter]キーを押すことでキャンバスに反映されます。文字サイズと字体の変更が可能です。Enterボタン(Alt+Enter)を押すと入力フォーム内のすべての文字が現在指定されている色・サイズ・字体で（再）描画されます。特定文字の色やスタイルを変更したい場合はフォーム内改行やスペース調整を駆使してください()
	⑤Position
		画面の右上、左上、右下、左下の4か所から張り付ける場所を選択できます。
	⑥Peta!! ボタン
		キャンバスの内容が画面にPeta!!されます。




3.付箋説明(PetaView.exe)
	付箋はタスクバーには表示されません。慌てないでください（？）
	付箋上にマウスカーソルを合わせてCtrlキーを押すと最前面に表示されます。タスクバーの下に隠れてしまったときにご利用ください。
	終了方法は1.で述べた通りです。




4.おまけ
	peta.bmpを差し替えてPetaView.exeを起動することで好きな画像を張り付けることができます。（表示位置は自動調整されます。）作業中も推しの画像を眺めていたいあなたに。
